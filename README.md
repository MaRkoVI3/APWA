# APWA — AI Powered Wellbeing Assistant

A simple student project — a mobile wellbeing app powered by Claude AI.

## How to Run

```bash
cd backend
pip install -r requirements.txt
python main.py
```

Then visit http://localhost:8000 in your browser.

## Tech Stack

- **Python** - The programming language for the backend
- **FastAPI** - Web framework that turns Python into a server
- **Anthropic Claude** - AI API that powers the chat feature
- **SQLite** - Simple local database for storing mood entries
- **HTML/CSS/JavaScript** - Frontend that runs in a mobile browser
